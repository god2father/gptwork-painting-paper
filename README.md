# 世界名画剪纸拆解

一个将世界名画转化为可交互剪纸舞台的网页实验。

[在线体验](https://god2father.github.io/gptwork-painting-paper/)

## 项目目的与灵感来源

这是一次 **GPT Work 视频理解与创意还原测试**：向 GPT Work 提供一段展示最终效果的视频，让它理解其中的视觉语言、运动节奏和交互想法，再独立实现为可运行的网页。

项目的剪纸名画表现方式由参考视频启发，但不追求逐帧复制。参考视频只作为分析输入，不作为仓库内容或网页素材发布；网页中的画作素材由 GPT 生成并进行独立分层处理。

## 项目简介

打开页面时，裁纸刀会沿纸张中线划开“裁”字，随后画作纸片沿各自轨迹逐层飞入并拼成完整画面。用户可以移动鼠标观察景深与视差，点击画面中的独立纸片查看说明和三维浮起效果，也可以一键拆开全部图层或切换不同作品。

目前包含：

- 约翰内斯·维米尔《戴珍珠耳环的少女》
- 扬·凡·艾克《阿尔诺芬妮夫妇像》

## 本地运行

```bash
npm ci
npm run dev
```

生产构建：

```bash
npm run build
```

主要技术：Vue 3、TypeScript、Vite、GSAP、Pinia 与 Three.js。
