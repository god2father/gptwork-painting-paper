# 验收标准

## 输入和分析

- `reference/reference-video.mp4` 可被 FFprobe 读取。
- `docs/video-analysis.md` 包含时间码、场景、图层、运动和材质分析。
- `manifests/timeline-analysis.json` 是有效 JSON。
- 分析区分“观察事实”和“实现推测”。

## 分层素材

- 第一幅名画有完整背景和至少 8 个独立前景/中景图层。
- 所有移动图层具有透明通道。
- 被遮挡背景不存在明显透明空洞。
- 提供图层总览预览。
- 存在 `docs/manual-review.md`，即使没有问题也要明确写“未发现待确认项”。

## 网站功能

- 首屏可看到完整组合的名画。
- 支持自动播放、暂停、重播、还原和时间轴跳转。
- 支持滚动拆解和鼠标视差。
- 支持点击图层查看信息。
- 使用配置文件驱动，不依赖某幅作品的硬编码。
- 不使用视频或 GIF 作为主要效果。

## 可访问性与性能

- 键盘可操作主要控制按钮。
- 有 `prefers-reduced-motion` 降级。
- 图片有合理的替代文本或图层描述。
- 1920×1080 桌面视口下交互流畅。
- 移动端不出现横向溢出和控制按钮不可点击问题。

## 工程质量

- `npm run typecheck` 通过。
- `npm run test` 通过。
- `npm run test:e2e` 通过。
- `npm run build` 通过。
- README 包含安装、运行、素材替换和新增作品方法。
