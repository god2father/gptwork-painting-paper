# 剪纸视觉系统规范

## 视觉关键词

手工编辑拼贴、真实纸片层次、博物馆印刷品、轻微复古、清晰而非重颗粒、平面材料形成 2.5D 空间。

## 背景纸

- 暖米白或浅灰无涂布纸。
- 纤维纹理必须细微，不能影响名画细节。
- 禁止科技蓝渐变、玻璃拟态和塑料 3D 材质。

## 图层边缘

- 主体外侧使用窄暖白纸边。
- 机器裁切感为主，局部允许轻微不规则。
- 同一作品内白边宽度需要按画面尺寸统一换算。
- 禁止粗大贴纸描边。

## 阴影

- 全局光源一致。
- 阴影向右下偏移。
- 低透明、柔和、短距离。
- 图层越靠前，阴影距离可略增加，但不得像悬浮卡片。

## 运动

- 纸片具有轻微惯性、落位回弹和错峰。
- 避免所有元素使用完全相同的缓动。
- 拆解时以“构图关系清晰”为先，不以夸张飞散为先。
- 鼠标视差需要克制，避免眩晕。

## 清晰度

- 纹理不等于噪点；禁止重胶片颗粒。
- 原画主体细节优先于纸张纹理。
- 透明图层按实际显示尺寸导出 1.5–2 倍分辨率。

## 色彩

- 尽量保留原画颜色。
- 纸张化处理只轻微改变对比度和边缘。
- UI 使用中性色，不能抢夺作品注意力。
